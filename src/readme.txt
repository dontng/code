# 作者序
    我不断反复地问自己一个问题：这本书是讲什么的？

    “带领大家通过一段信息技术革命的旅程重新感触现代计算机技术”
    “其实是讲述计算机如何工作的”

    明喻与暗喻是文学描述中精妙的辅助手段，但它们常常掩盖了科学技术的真正光芒

    “人们不愿意了解计算机如何工作”，这句话我赞同
    因为就我个人而言，在不经意间体会到理解事物如何运作这一过程是多么有趣，但这并不意味着我对世上一切事物的运作机制都感兴趣

    例子：存储器（storage）与内存（memory）的区别？

    市场上考量个人计算机的存储性能，最主要的就是这两个概念
    内存和存储的区别其实是在逻辑层面上的，体现着计算机体系结构的实际需求与存储器客观性能之间的矛盾，
    简单的说，就是我们找不到一种同时具备这两种存储器所有优点的存储媒介
